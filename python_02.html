Haaste Python_from_Scratch_00

Ensimmäisessä haasteessa kunnioitamme perinteitä tekemällä "Hello, world!" -ohjelman. Tässä haasteessa emme vielä käytä pelaajaoppimisen metodia, vaan ratkaisemme haasteen ohjeita seuraamalla. Haaste on varmasti helppo ratkaista. Tarkoituksena on tutustua ohjelmointiympäristöön ja sen toimintaan.

Ohjeet

  1. Kirjoita # KIRJOITA TÄMÄN RIVIN ALAPUOLELLE OMA KOODISI -tekstin alapuolelle seuraava lause:

    print("Hello, world!")

  2. Aja ohjelma ► Run -kuvaketta klikkaamalla.

Haaste Python_from_Scratch_02

Tässä haasteessa on tehtävänä saada Pycat liukumaan mustaan kissan muotoiseen aukkoon. Tässä tehtävässä sovelletaan pelaajaoppimisen metodia oikean komennon ja oikeiden koordinaattien löytämiseen.

Ohjeet

  1. Tässä tehtävässä tarvitaan koordinaatteja apuun. Laita oikeat koordinaatit oikeaan lauseeseen (löydät listan turtle-komennoista otsikolla "Komentotaulukko" tältä sivulta alempaa).
  2. Kirjoita # KIRJOITA TÄMÄN RIVIN ALAPUOLELLE OMA KOODISI -tekstin alapuolelle.
  3. Aja ohjelma ► Run -kuvaketta klikkaamalla.
