Haaste Python_from_Scratch_03

Tässä haasteessa opitaan, kuinka Pycat saadaan tanssimaan ikuisesti. Opitaan, kuinka Pycat saadaan liikkumaan edestakaisin . Opitaan käyttämään ikuista silmukkaa.

Tässä haasteessa käytetään pelaajaoppimisen metodia. Ratkaisun voi löytää yrityksen ja erehdyksen kautta. Yrityksen ja erehdyksen menetelmä on ongelmanratkaisun perusmenetelmiä.

Ohjeet

  1. Tarvitaan lause, jolla saadaan hahmo (Pycat) liikkumaan eteenpäin (ja taaksepäin). Sen voi löytää sivun alalaidasta Komentotaulukosta tai katsomalla Lause-teoriaosuudesta.
  2. Tarvitaan lause, jolla ohjelma saadaan odottamaan hetken. Liike on niin nopea, että sitä voi olla vaikea havaita. Tähän on olemassa oma komentonsa, joka löytyy Python dokumentaatiosta.
  3. Koodi pitää sijoittaa silmukkaan niin, että liike jatkuu kunnes ohjelma pysäytetään.
  4. Kirjoita # KIRJOITA TÄMÄN RIVIN ALAPUOLELLE OMA KOODISI -tekstin alapuolelle.
  5. Aja ohjelma ► Run -kuvaketta klikkaamalla.

Haaste Python_from_Scratch_04

Tässä haasteessa opitaan tekemään animaatio. Tämä on tähän mennessä haastavin tehtävä.

Tässä haasteessa käytetään pelaajaoppimisen metodia. Turhaantuminen ja ärtymyksen tunne ovat oleellinen osa oppimisprosessia vaikeassa haasteessa.

Tehtävänä on animoida Pycat kävelemään vasemmalta oikealle näyttämöllä.

Tämän tehtävän suoritettua ymmärtää periaatteen, miten animaatioita voi tehdä Pythonilla.

Ohjeet

  1. Käytetään tähän mennessä opittuja taitoja tehtävän ratkaisemisessa
  2. Vihje: Tässä tehtävässä käytetään kahta hahmoa. Löydät niiden nimet # KIRJOITA TÄMÄN RIVIN ALAPUOLELLE OMA KOODISI -tekstin yläpuolella olevaa koodia tutkimalla. Niitä toistuvasti vaihtamalla, näyttämällä ja piilottamalla sekä liikuttamalla samalla hahmoa saadaan hahmo animoitua.
  3. Kirjoita # KIRJOITA TÄMÄN RIVIN ALAPUOLELLE OMA KOODISI -tekstin alapuolelle.
  4. Aja ohjelma ► Run -kuvaketta klikkaamalla.

Haaste 04

Haaste Python_from_Scratch_05

Tässä haasteessa opitaan lisäämään interaktiivisuutta projektiin. Haasteena on tehdä ohjelma, joka kysyy kolme kertaa nimeä ja hahmo (Pycat) kirjoittaa joka kerran jälkeen nimen näyttämölle.

Ohjeet

  1. Tarvitaan komento, jolla käyttäjältä kysytään nimi ja saadaan hahmo kirjoittamaan se näyttämölle. Dokumentaatiosta löytyy tarkoitukseen sopiva.
  2. Tarvitaan komento, jolla saadaan kirjoitus näkymään halutun ajan näyttämöllä.
  3. Tarvitaan komento, jolla kirjoitus saadaan pyyhittyä ennen uuden ilmestymistä. Dokumentaatiosta löytyy tarkoitukseen sopiva.
  4. Tehty koodi pitää laittaa silmukkaan niin, että se toistuu 3 kertaa.
  5. Kirjoita # KIRJOITA TÄMÄN RIVIN ALAPUOLELLE OMA KOODISI -tekstin alapuolelle.
  6. Aja ohjelma ► Run -kuvaketta klikkaamalla.
